 사용자가 초반에 선호하는 식단을 추천을 어떤식으로 어디까지 고려하여 추천해줄 지 고민해보며, 팀원들과 어떻게 하는 것이 좋을지 이야기를 나누었습니다.
 프로젝트 내에서 같은 방향성을 바라보고 있는 것이 생각보다 쉽지 않지만, 이 시간들 덕분에 나중에 백/프론트 나누어 개발 시작을 하면 원활하게 할 수 있지 않을까?! 생각이 들었습니다!
